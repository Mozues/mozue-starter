# Mozue Starter

Starter code for the Mozue personal website using Next.js, TypeScript, Tailwind CSS, and Vercel.

## 1) Install dependencies
```bash
npm install
```

## 2) Run locally
```bash
npm run dev
```

## 3) Open in browser
```bash
http://localhost:3000
```

## 4) Deploy to Vercel
- Push this project to GitHub
- Import the repo into Vercel
- Add your custom domain in Vercel
- Update DNS in GoDaddy

## What to replace first
- Replace the placeholder image circles with your real logos/images
- Update the `Ventures` page links
- Update footer year if needed
- Refine fonts later if you want Google Fonts
